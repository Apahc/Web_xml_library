*Быстрый запуск (одной командой)*

1. Открой терминал в корне проекта (TreeLink/TreeLink_v1)
2. Выполни:

'''
docker compose up --build
'''

3. Дождись завершения сборки и запуска контейнеров.
4. Открой в браузере:
http://localhost:8080

Готово — интерфейс XML Tree Viewer работает, загрузка XML и просмотр структуры доступны.
